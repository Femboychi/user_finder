## Описание программы

GitHub User Finder — это GUI‑приложение для поиска пользователей GitHub с использованием официального API. Программа позволяет:
* искать пользователей GitHub по имени;
* просматривать результаты поиска;
* добавлять пользователей в избранное;
* сохранять и загружать список избранных пользователей в формате JSON;
* использовать Git для контроля версий данных.

Программа полезна для разработчиков, рекрутеров, HR‑специалистов и всех, кто хочет отслеживать интересных пользователей GitHub.

## Требования для запуска

Что нужно установить на компьютер:
* Python 3.8 или выше;
* библиотеки: `pip install requests`.

---

## Как запустить

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Femboychi/user_finder.git
```
2. Перейдите в директорию проекта:

```bash
cd user_finder
```
3. Установите зависимости:

```bash
pip install requests
```
4. Запустите приложение:

```bash
python main.py
```
---

## Как пользоваться
При запуске откроется окно приложения с полем ввода.

- Введите имя пользователя GitHub в поле «Имя пользователя GitHub».

- Нажмите кнопку «Найти» — результаты появятся в списке ниже.

Чтобы добавить пользователя в избранное:

- выберите пользователя из списка результатов;

- нажмите «Добавить в избранное».

Для просмотра избранного нажмите «Показать избранное».

Для сохранения избранного нажмите «Сохранить избранное» (данные сохранятся в файл favorites.json).

Для загрузки ранее сохранённого избранного нажмите «Загрузить избранное» (приложение прочитает данные из favorites.json и отобразит их в списке).
