## Описание программы

Weather Diary — это GUI‑приложение для ведения дневника погоды. Программа позволяет:
* добавлять записи о погоде с указанием даты, температуры, описания и наличия осадков;
* просматривать все сохранённые записи в табличном виде;
* фильтровать записи по дате и температуре;
* сохранять данные в формате JSON и загружать их из файла;
* использовать Git для контроля версий данных.

Программа полезна для любителей метеорологии, садоводов, путешественников и всех, кто хочет отслеживать погодные условия в своём регионе.

## Требования для запуска

Что нужно установить на компьютер:
* Python 3.8 или выше;
* стандартные библиотеки Python (tkinter, json, datetime), которые входят в стандартную поставку Python.

---

## Как запустить

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Femboychi/itog.git
```
2. Перейдите в директорию проекта:
```bash
cd itog
```
3. Запустите приложение:
```bash
python main.py
```

## Как пользоваться
- При запуске откроется окно приложения с полями ввода:

- Дата (формат YYYY‑MM‑DD);

- Температура (°C);

- Описание погоды (текст);

- Осадки (флажок: отмечено — «да», не отмечено — «нет»).

Заполните все поля.

- Нажмите кнопку «Добавить запись» — запись появится в таблице ниже.

Для фильтрации:

- введите дату в поле «Фильтр по дате» (формат YYYY‑MM‑DD);

- укажите минимальную температуру в поле «Фильтр по температуре (>)» (числовое значение);

- нажмите «Применить фильтры» — в таблице отобразятся записи, соответствующие условиям.

Чтобы сбросить фильтры и вернуть полный список записей, нажмите «Сбросить фильтры».

Для сохранения всех текущих записей в файл JSON нажмите «Сохранить в JSON» (данные сохранятся в файл weather_records.json).

Для загрузки ранее сохранённых записей из файла JSON нажмите «Загрузить из JSON» (приложение прочитает данные из weather_records.json и отобразит их в таблице).
